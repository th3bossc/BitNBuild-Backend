

## Usage 
### Frontend
- Navigate to /frontend and run `npm start`

### Backend
Start server - `python3 manage.py runserver`
- REGISTER          POST                    /api/auth/register
- LOGIN             POST                    /api/auth/login
- WATCHLIST         GET, POST, DELETE 
- HISTORY           GET, POST, DELETE
- PROFILE           GET, PUT

- FIND A MOVIE      POST

- SELECT BY GENRE   GET

- POPULAR MOVIES    GET
- UPCOMING MOVIES   GET
